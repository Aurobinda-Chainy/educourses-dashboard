# EduCourses Dashboard

EduCourses Dashboard is a responsive and user-friendly web-based admin interface designed to help manage educational content efficiently. It provides clear insights into courses, students, instructors, and batch activity through a clean dashboard layout.

The project uses modular templates for easy scalability and custom styling that works across all screen sizes, including mobile devices.

---

## Features

- Dashboard displaying key metrics at a glance
- Course management table
- Student list with enrollment status
- Instructor directory with profile preview
- Sidebar navigation layout
- Light/Dark theme toggle
- Fully responsive and mobile-friendly design

---

## Technologies Used

| Technology | Purpose |
|----------|---------|
| HTML5 | Structure of the application |
| CSS3 / SCSS | Styling and theme customization |
| Bootstrap | UI layout and components |
| JavaScript (Vanilla) | Dynamic interactions and DOM updates |
| Nunjucks | Templating engine for reusable UI components |
| Vite | Development server and bundler |
